Why?

핵심 기술 결정

왜 Gemini?
오디오 직접 입력 + 유료 등록 (학습 제외)
Whisper+Claude 2단계 대비 1콜로 전사+요약 해결. 카드 등록으로 데이터 학습 제외 보장.
왜 Vercel?
서버 관리 0, 무료
git push만 하면 자동 배포. 서버리스 함수 + Blob 저장소 무료 포함.
왜 청크 분할?
Vercel 4.5MB 본문 한도
1시간 녹음 ≈ 14MB. 3.5MB씩 4조각으로 나눠 업로드 후 서버에서 결합.
왜 PWA?
앱 스토어 없이 설치
URL 접속 → 홈 화면 추가 = 끝. iOS/Android 모두 동일하게 동작.
